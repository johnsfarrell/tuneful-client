An excerpt:

To run locally, clone the repository and follow the directions below:

```bash
cd client
npm install
npm run start
```

| Command         | Action                     |
| --------------- | -------------------------- |
| `npm run start` | Start the server           |
| `npm run build` | Build a production version |
| `npm run test`  | Run the test suite         |
